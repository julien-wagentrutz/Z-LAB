Comment jouer à Z-LAB ?

Bienvenue dans le meilleur jeu combinant labyrinthes et zombies : Z-LAB. En tant qu'aventurier, vous avez pour mission de récolter votre équipement, puis fuir et évitant de vous faire manger par un zombie !

Le but du jeu

Récoltez tous les items dont vous avez besoin : ... puis foncez vers la sortie en évitant les zombies affamés qui vous cherchent ! Attention, vous risquez de vous retrouver dans le noir à un moment, j'espère que cela ne vous effraie pas...

Les touches

Pour avancer, vous aurez besoin des flèches sur votre clavier.

Utilisez la touche Shift en vous déplaçant vers un ennemi pour donner un coup d'épée.

La barre d'espace vous permettra de poser une bombe, utile pour détruire les murs abimés.

L'équipement

L'épée : elle vous permet de combattre les zombies. Pour cela, utilisez la touche Shift.

La bombe : récupérez la puis déposez la devant un mur abimé avec la barre d'espace pour le détruire.

La clef : essentille pour s'enfuir, elle permet d'ouvir la porte de sortie.
